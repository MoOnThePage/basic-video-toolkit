# Basic Video Toolkit

A simple, modular Python tool to learn video processing with OpenCV.

## Features (Planned)
- [ ] Play saved video files
- [ ] Preview Webcam feed
- [ ] Save video with 's' key
- [ ] Toggle motion detection
- [ ] Auto-save on motion

## Requirements
- Python 3.xx
- OpenCV

## Usage
```bash
python main.py
```

## Project Structure
- `main,py`: Entry point with menu
- `video_recorder.py`: Webcam Capture, recording, motion detection
- `video_player.py`: Video Playback logic
- `outputs/`: Saved video
